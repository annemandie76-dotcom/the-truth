# JODAMS Online Daily

JODAMS Online Daily is a modern, responsive news website focused on political and economic stories.  
The website is built as a static site and later enhanced with Netlify CMS so the client can publish stories independently.

---

## 🚀 Features

- Responsive design (mobile, tablet, desktop)
- Modern green-themed header branding
- Homepage with latest news previews
- Stories page with full article pages
- Clickable news cards
- Contact section with social media icons
- Advertisement section for sponsored content
- Prepared for Netlify CMS integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (basic navigation toggle)
- Boxicons (icons)
- Google Fonts
- Netlify (hosting)
- Netlify CMS (planned)

## ▶️ How to Run Locally

1. Download or clone the repository
2. Open `index.html` in your browser
3. No server required (static website)

---

## 🌍 Hosting

This website is hosted for free using **Netlify**.  
It does not require a custom domain to work (a Netlify subdomain is used).

---

## ✍️ Content Management

Netlify CMS will be integrated to allow the client to:

- Add new stories
- Edit existing articles
- Publish content without touching code
